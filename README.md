## Getting Started

* Clone the repository:
    ```bash
    git clone https://github.com/yourusername/python-sandbox.git
    cd python-sandbox
    ```

* Run a script:
    ```bash
    python path/to/script.py
    ```

* Run tests:
    ```bash
    make test
    ```
